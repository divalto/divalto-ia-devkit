## Conventions

Chaque entree porte un identifiant unique (R-NNN) et une categorie :

| Categorie | Signification |
|-----------|---------------|
| BUG-SKILL | Bug dans un skill ou script Python |
| BUG-DOC | Documentation incorrecte ou manquante dans le skill |
| ENV | Specifique a l'environnement (chemin, config, utilisateur) |
| CLAUDE-TOOL | Limitation de Claude Code (Edit, Write, encodage) |
| SUGGESTION | Amelioration ou fonctionnalite souhaitee |

Severites :
- **CRITIQUE** : bloquant, pas de contournement
- **HAUTE** : contournement penible ou risque d'erreur
- **MOYENNE** : contournement facile
- **BASSE** : cosmetique ou ergonomie
